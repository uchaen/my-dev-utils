# Dev Utils

개발 업무에 필요한 유틸리티 모음 사이트

🔗 [https://uchaen.github.io/my-dev-utils/](https://uchaen.github.io/my-dev-utils/)



## 주요 기능

### 🔒 JWT Encoder
Header, Payload, Secret Key를 입력하여 JWT 토큰을 실시간으로 생성합니다.

### 🔐 JWT Decoder
JWT 토큰을 입력하면 Header, Payload, Signature를 분리하여 디코딩 결과를 보여줍니다.

### 🔒 SHA256 암호화
텍스트를 SHA256 해시로 변환합니다. Hex 또는 Base64 형식으로 출력할 수 있습니다.

### 📝 JSON Beautifier
JSON을 포맷팅하거나 압축합니다. 구문 강조와 인덴트 컬러링 기능을 제공합니다.

### ⏰ Timestamp 변환기
Unix Timestamp와 날짜/시간 간의 실시간 변환을 수행합니다. 현재 시간(KST)도 확인할 수 있습니다.

### 📏 Text Size 계산기
텍스트의 글자수, 바이트 크기, 문자 유형을 실시간으로 계산합니다.

### 🔍 Text Diff 비교
두 텍스트의 차이점을 문자 단위로 실시간으로 비교하여 추가/삭제/수정된 부분을 표시합니다.
